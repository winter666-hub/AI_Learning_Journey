# AI Learning Journey

이 저장소는 머신러닝과 AI를 학습하며 정리한 노트,
알고리즘 구현 코드, 그리고 작은 프로젝트들을 기록한 공간입니다.

단순히 코드를 따라치는 것이 아니라,
개념을 이해하고 직접 구현하며 성장하는 과정을 담고 있습니다.

---

## Learning Goals

- 기본 개념과 원리를 정확히 이해하기
- 알고리즘이 왜 필요한지, 어떻게 동작하는지 설명할 수 있도록 정리
- 머신러닝을 실제 문제와 서비스에 적용하는 경험 쌓기
- 꾸준히 기록하며 성장하기
